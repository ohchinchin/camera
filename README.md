# iPhone 7 Plus 定点観測カメラ (Discord送信)

iPhone 7 Plus (iOS 15) のブラウザを定点カメラとして使用し、定期的に Discord へ画像を送信するシステムです。

## セットアップ手順

1. **GitHub リポジトリの作成**
   - GitHub で新しいリポジトリを作成し、このコードをプッシュしてください。

2. **Vercel へのデプロイ**
   - Vercel でプロジェクトをインポートします。
   - **Environment Variables** に以下を設定してください。
     - `DISCORD_WEBHOOK_URL`: Discord のチャンネル設定から取得した Webhook URL。

3. **iPhone での利用**
   - デプロイされた URL を iPhone の Safari で開きます。
   - カメラの許可を求められるので「許可」します。
   - 「撮影開始」ボタンを押します。

## 注意事項 (iOS 15 向け)

- **スリープ防止:** `NoSleep.js` を使用していますが、iPhone の設定でも「画面表示と明るさ」→「自動ロック」を「なし」に設定することを推奨します。
- **電源:** カメラを常時起動し、Wi-Fi通信を行うため、常に充電器に接続した状態で使用してください。
- **フォアグラウンド:** ブラウザがバックグラウンド（他のアプリを開く、画面を閉じる）になると動作が停止します。
